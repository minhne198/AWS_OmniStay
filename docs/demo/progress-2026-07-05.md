# Nhật ký công việc 05/07/2026

## Mục tiêu đã xử lý

Dự án AWS OmniStay đã được dựng theo hướng code trước, cấu hình AWS Console sau.
Mentor yêu cầu khi cấu hình AWS phải chụp màn hình, nên phần code và tài liệu
hướng dẫn đã được chuẩn bị trước để ngày mai có thể tạo dịch vụ AWS trên web
console và chụp bằng chứng theo checklist.

## Backend

- Tạo solution `backend/AWSOmniStay.slnx`.
- Tạo ASP.NET Core Web API tại `backend/src/HotelBooking.Api`.
- Tạo các endpoint chính:
  - `GET /health`.
  - `GET /health/aws`.
  - `GET /api/hotels/search`.
  - `GET /api/hotels/{id}`.
  - `GET /api/hotels/{id}/rooms`.
  - `POST /api/bookings`.
  - `GET /api/bookings/{code}`.
- Tạo model dữ liệu `Hotel`, `RoomType`, `Booking`.
- Tạo EF Core `HotelBookingDbContext`.
- Seed dữ liệu mẫu gồm 10 khách sạn và 30 loại phòng.
- Local/test mặc định dùng EF Core InMemory để chạy nhanh không cần AWS.
- Thêm cấu hình AWS production dùng Amazon RDS MySQL khi set:
  - `Database__Provider=MySql`.
  - `ConnectionStrings__HotelBookingDb=...`.
- Thêm provider `MySql.EntityFrameworkCore`.
- Thêm Redis cache bằng `StackExchange.Redis`.
- Tạo `RedisConnectionFactory`, `IHotelSearchCache`, `RedisHotelSearchCache`.
- Tạo `CachedHotelBookingService` bọc service EF Core.
- Search khách sạn ghi log cache `MISS` lần đầu và `HIT` khi có Redis cache.
- Sau khi tạo booking thành công, cache search được xóa để tránh dữ liệu phòng
  trống bị cũ.
- Endpoint `/health/aws` trả thông tin runtime để chụp bằng chứng:
  - environment.
  - region.
  - database provider.
  - Redis configured/connected.
  - cache TTL.
  - S3 frontend bucket.
  - CloudFront domain.
  - API base path.

## Frontend

- Tạo frontend static HTML/CSS/JavaScript trong `frontend/`.
- Frontend dùng API thật từ backend, không dùng mockup.
- Tạo luồng tìm phòng, xem chi tiết khách sạn, chọn loại phòng và đặt phòng.
- Tạo `frontend/src/api.js` để gọi backend API.
- Tạo `frontend/src/app.js` xử lý state UI và form booking.
- Tạo `frontend/src/styles.css` cho giao diện responsive.
- Tạo `frontend/public/config.js` cho local API base:
  - `http://localhost:5010/api`.
- Tạo `frontend/public/config.aws.example.js` cho CloudFront production:
  - `API_BASE: "/api"`.
