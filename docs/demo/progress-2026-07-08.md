# Nhật ký công việc 08/07/2026

## Mục tiêu đã xử lý

Dự án AWS OmniStay đã chuyển sang giai đoạn chuẩn bị giao diện, tài liệu triển khai AWS Console
và bằng chứng ảnh chụp dịch vụ. Sau mốc 05/07/2026, phần code backend/frontend local vẫn được giữ
ổn định; trọng tâm mới là đưa prototype giao diện vào dự án, chuẩn bị tài liệu hướng dẫn tạo dịch vụ
AWS từng bước và phân công việc tạo/chụp ảnh AWS Console.

Nhật ký này chỉ ghi nhận tiến độ công việc và các resource/tài liệu cần dùng cho demo. Không ghi
secret, password, access key, token hoặc endpoint nhạy cảm.

## Frontend prototype

- Đưa bộ giao diện do AI/Stitch tạo vào dự án tại `frontend/public/prototype`.
- Giữ nguyên frontend cũ đang gọi API thật tại `frontend/public/index.html` để tránh làm hỏng luồng
  local hiện có.
- Tạo trang mục lục prototype tại:

  - `booking-confirmation.html` - trang xác nhận đặt phòng.
  - `booking-lookup.html` - trang tra cứu/lịch sử booking.
  - `system-status.html` - trang trạng thái hệ thống AWS demo.
- Copy ảnh preview gốc vào:
  - `frontend/public/prototype/screens`.
- Thêm tài liệu map route prototype tại:
  - `frontend/public/prototype/README.md`.
- Lưu design note của bộ giao diện tại:

- Trạng thái frontend hiện tại:

## AWS services

- Việc tạo các dịch vụ AWS trên Console đã được giao cho bạn làm.
- Coi như đã hoàn tất bước tạo dịch vụ và chụp ảnh bằng chứng ban đầu trên AWS Console.
- Các dịch vụ nằm trong phạm vi đã hướng dẫn/chụp ảnh:
  - AWS Budget.
  - VPC `omnistay-vpc`.
  - 6 subnets:
    - `omnistay-public-a`.
    - `omnistay-public-b`.
    - `omnistay-app-a`.
    - `omnistay-app-b`.
    - `omnistay-data-a`.
    - `omnistay-data-b`.
  - Internet Gateway.
  - NAT Gateways.
  - Public/private route tables.
  - Security Groups:
    - `SG-ALB`.
    - `SG-EC2`.
    - `SG-RDS`.
    - `SG-Redis`.
  - Amazon RDS MySQL `omnistay-mysql`.
  - Amazon ElastiCache Valkey/Redis `omnistay-cache`.
  - IAM Role `EC2-HotelAPI-Role`.
  - Secrets Manager và Parameter Store.
  - S3 frontend/artifact buckets.
  - Target Group `omnistay-api-tg`.
  - Application Load Balancer `omnistay-alb`.
  - Launch Template `omnistay-api-template`.
  - Auto Scaling Group `omnistay-api-asg`.
  - CloudFront distribution với Origin Access Control.
  - AWS WAF cho CloudFront.
  - SNS topic `omnistay-alerts`.
  - CloudWatch dashboard và alarms.
- Ghi chú quan trọng:
  - Mốc này chỉ xác nhận bước tạo resource và chụp ảnh Console ban đầu.
  - Chưa xác nhận backend đã chạy trên EC2.
  - Chưa xác nhận frontend đã deploy lên S3/CloudFront.
  - Chưa xác nhận flow end-to-end CloudFront -> ALB -> EC2 -> RDS/Redis đã chạy thật.
