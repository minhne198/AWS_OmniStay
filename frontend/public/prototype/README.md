# OmniStay Frontend Prototype

This folder contains static visual prototype pages copied from `stitch_skystay_hotel_booking`.

## Pages

| Prototype file | Intended app route | Purpose |
| --- | --- | --- |
| `home.html` | `/` | Home and hotel search entry point |
| `search.html` | `/search` | Search results and filters |
| `hotel-detail.html` | `/hotels/:id` | Hotel detail and room selection |
| `booking.html` | `/booking` | Booking form and trip summary |
| `booking-confirmation.html` | `/booking/success` | Booking success and code display |
| `booking-lookup.html` | `/lookup` | Booking lookup/history by booking code |
| `system-status.html` | `/system-status` | AWS demo runtime status |

## Preview

Open `index.html` in this folder to browse all prototype pages and screenshots.

These pages are static previews. The API-connected app still lives at `frontend/public/index.html`.
