Dịch vụ cốt lõi bắt buộc

Dịch vụ Vai trò trong hệ thống Đơn giá tham khảo Ưu điểm Ghi chú sử dụng
Amazon VPC, Subnets, Route Tables, Internet Gateway Tạo mạng riêng, chia public/private subnet, cho ALB ra internet. $0 cho VPC/subnet/route/IGW cơ bản. Cô lập hệ thống, dễ thiết kế public/private layer đúng kiến trúc cloud. Chi phí phát sinh từ NAT Gateway, VPC endpoints, public IPv4 hoặc data transfer.
Security Groups Firewall theo từng layer: ALB, EC2, RDS, Redis/Valkey. $0 cho rule/security group. Quản lý inbound/outbound rõ ràng, dễ giải thích trong demo bảo mật. Chỉ mở EC2 port 8080 từ SG của ALB; RDS/Redis chỉ mở từ SG của EC2.
Application Load Balancer Nhận HTTP/HTTPS từ người dùng và chia tải vào EC2 trong Auto Scaling Group. $0.0252/giờ + $0.008/LCU-giờ. Health check, routing, scale cùng ASG, demo HA khi terminate 1 EC2. Cần target group health check `/health`; ALB tốn phí theo giờ ngay cả khi ít traffic.
Amazon EC2 Chạy backend ASP.NET Core API trên Linux. t3.micro Linux: $0.0132/giờ. Dễ demo server truyền thống, cài .NET runtime, quan sát CPU và scale-out. Chưa gồm EBS root volume, public IPv4, data transfer. Nên dùng IAM role thay access key.
Amazon EBS Ổ đĩa gốc cho EC2 và nơi lưu artifact/app runtime trên instance. gp3: $0.096/GB-tháng; IOPS/throughput vượt mức cơ bản tính riêng. Lưu trữ bền vững hơn instance store, dễ snapshot/backup khi cần. Demo nên dùng root volume nhỏ, xóa volume khi terminate nếu không cần giữ.
EC2 Auto Scaling Group Tự tăng/giảm số EC2 dựa trên CPU hoặc health check. $0 cho ASG; chỉ trả tiền EC2/ALB/CloudWatch liên quan. Chứng minh auto scaling khi chạy Locust load test. Plan đề xuất min=1, desired=1, max=4 cho demo.
Amazon RDS for MySQL Database chính lưu user, hotel, room type, booking và revenue data. db.t3.micro Single-AZ MySQL: $0.026/giờ. Quản lý database dễ hơn tự cài MySQL trên EC2; có backup, snapshot, monitoring. Single-AZ để tiết kiệm khi demo; Multi-AZ dùng để thuyết minh HA thật nhưng tốn thêm chi phí.
Amazon ElastiCache Redis OSS / Valkey Cache kết quả tìm kiếm khách sạn và giảm truy vấn lặp lại vào RDS. Valkey cache.t3.micro: $0.020/giờ; Redis OSS cache.t3.micro: $0.025/giờ. Độ trễ thấp, demo cache hit/miss rõ ràng, giảm tải database. Nên dùng TTL ngắn, ví dụ 5 phút, cho kết quả search phổ biến.
Amazon S3 Lưu frontend static files và có thể lưu ảnh khách sạn. $0.025/GB-tháng S3 Standard first 50 TB; PUT/LIST $0.005/1,000; GET $0.004/10,000. Rẻ, bền, không cần server web riêng, kết hợp CloudFront tốt. Dùng private bucket + Block Public Access; không bật Static Website Hosting nếu dùng OAC.
Amazon CloudFront CDN cho frontend từ S3 và có thể forward `/api/*` về ALB. $0.120/GB data out Asia first 10 TB; HTTPS GET/HEAD $0.012/10,000. HTTPS, cache static file, giảm latency, bảo vệ S3 origin. CloudFront free tier có 1 TB data transfer out/tháng nếu đủ điều kiện tài khoản.
CloudFront Origin Access Control Cho CloudFront đọc private S3 bucket an toàn. $0 phí riêng. Không cần public S3 bucket; phù hợp bảo mật và báo cáo kiến trúc. Origin phải là S3 REST endpoint, không dùng S3 website endpoint.
IAM Roles and Policies Cấp quyền cho EC2 đọc S3, SSM, CloudWatch mà không dùng access key. $0. Giảm rủi ro lộ key; dùng chuẩn AWS cho workload trên cloud. Cần role riêng cho EC2, CDK deploy và quyền tối thiểu khi có thể.
SSM Parameter Store Lưu config/secret runtime như connection string, JWT secret nếu dùng SecureString. Standard parameters: $0; Advanced parameter khoảng $0.05/parameter-tháng. Dễ dùng, tích hợp IAM/KMS, phù hợp demo tiết kiệm. Đủ cho phiên bản demo; dùng Secrets Manager nếu cần rotation đầy đủ.
AWS Secrets Manager Lưu DB password/JWT secret và hỗ trợ rotation. $0.40/secret-tháng + $0.05/10,000 API requests. Quản lý secret chuyên nghiệp, audit/rotation tốt hơn file `.env`. Có thể dùng thay SSM cho DB password nếu muốn trình bày bảo mật nâng cao.
Amazon CloudWatch Logs, metrics, alarms và dashboard cho ALB, EC2, ASG, RDS, cache. Logs ingest $0.70/GB; standard alarm $0.10/alarm-tháng; custom metric $0.30/metric-tháng. Có bằng chứng demo: CPU tăng, ASG scale, ALB latency/5xx, log API. Đặt retention ngắn cho log group để tiết kiệm.
AWS CDK + AWS CloudFormation Định nghĩa và deploy hạ tầng bằng code TypeScript. $0 phí riêng cho CDK/CloudFormation cơ bản; trả tiền resource tạo ra. Lặp lại được demo, dễ xóa tạo lại, thể hiện Infrastructure as Code. CDK synth/deploy tạo CloudFormation stacks.

Dịch vụ điều kiện và tùy chọn

Dịch vụ Khi nào dùng Đơn giá tham khảo Ưu điểm Khuyến nghị cho đồ án
NAT Gateway EC2 nằm private subnet nhưng cần tải package, gọi S3/SSM/CloudWatch qua internet. $0.059/giờ + $0.059/GB processed. Cấu hình đơn giản, đúng production pattern. Tốn phí theo giờ. Chỉ bật khi cần demo private subnet đúng chuẩn, xóa sau lab.
VPC Endpoints Thay NAT cho kết nối private tới S3, SSM, CloudWatch Logs. Interface endpoint $0.013/endpoint-giờ + $0.01/GB; S3 Gateway endpoint $0. Bảo mật hơn NAT, traffic không ra public internet. Nếu muốn tiết kiệm, dùng S3 Gateway endpoint và chỉ tạo interface endpoints thật cần thiết.
Public IPv4 Address Khi EC2 public subnet hoặc Elastic IP được gán/bỏ trong tài khoản. $0.005/giờ/IP cho in-use hoặc idle public IPv4. Đơn giản cho demo public EC2 sau ALB. Không mở SSH public; nếu dùng public subnet, chỉ cho inbound từ ALB.
Route 53 Gắn domain riêng cho CloudFront/API. $0.50/hosted zone-tháng first 25 hosted zones; standard query $0.40/1M queries. Domain đẹp, dễ demo production-like. Tùy chọn. Có thể dùng CloudFront domain mặc định để tiết kiệm.
AWS Certificate Manager Cấp TLS certificate cho CloudFront/ALB khi dùng domain riêng. $0 cho public certificate dùng với AWS integrated services. Tự gia hạn, HTTPS đúng chuẩn. Bắt buộc nếu dùng domain riêng HTTPS; không dùng Private CA cho demo.
Amazon SES Gửi email xác nhận booking. $0.0001/email recipient, tương đương $0.10/1,000. Tăng tính thực tế cho sản phẩm, có thể demo email confirmation. Tùy chọn nâng cao; cần verify sender/domain trước khi gửi thật.
AWS Budgets Cảnh báo chi phí AWS trước khi vượt mức demo. Budget notification $0; action-enabled budget có thể $0.10/budget-day. Giảm rủi ro phát sinh hóa đơn, rất nên có trong đồ án. Tạo budget 10 USD, cảnh báo khi forecast vượt 5 USD.

Thứ tự ưu tiên khi triển khai

  • Giai đoạn local: chưa cần AWS, chỉ tạo backend `/health`, database model và frontend tĩnh.
  • Giai đoạn cloud tối thiểu: S3, CloudFront, ALB, EC2, RDS, ElastiCache, IAM, CloudWatch.
  • Giai đoạn demo HA/scale: Auto Scaling Group, CloudWatch alarms, Locust load test.
  • Giai đoạn production-like: NAT hoặc VPC endpoints, Route 53, ACM, Secrets Manager, SES.
